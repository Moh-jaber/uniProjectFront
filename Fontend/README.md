## Quick Start / Frontend

- **Prerequisites:** Install Node.js (recommended v16+). Verify with:

```bash
node --version
npm --version
```

- **Clone the repo:**

```bash
git clone <repo-url>
cd uniProjectFront/Fontend
```

- **Install dependencies:**

```bash
npm install
```

- **Run development server:**

```bash
npm run dev
```

This starts Vite's dev server (default: http://localhost:5173). Open the address shown in your terminal.
